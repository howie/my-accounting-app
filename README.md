# My Accounting Book (MyAB)

A local-first, double-entry personal finance accounting system inspired by [MyAB 我的記帳簿](https://www.devon.riceball.net/).

## Features

### Core Accounting

- **Multiple Ledgers** - Create and manage multiple accounting books
- **Double-Entry Bookkeeping** - Full support for Asset, Liability, Income, and Expense accounts
- **Transaction Management** - Record expenses, income, and transfers with strict validation
- **Hierarchical Accounts** - Support for nested account structures (e.g., Assets > Bank > Checking)

### User Interface

- **Dashboard** - Overview with total assets, income vs expenses charts, and monthly trends
- **Sidebar Navigation** - Hierarchical account tree for quick access
- **Account Transaction List** - View transactions with pagination and filtering
- **Mobile Responsive** - Hamburger menu for mobile devices
- **i18n Support** - Multi-language support (Traditional Chinese, English)

### Settings & Account Management

- Account CRUD with parent-child relationships
- Drag-and-drop account ordering
- Dark/Light mode toggle
- Language preferences
