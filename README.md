# LedgerOne

**Professional Double-Entry Accounting for the Modern Era**

LedgerOne is a local-first, privacy-focused personal accounting system built to bring professional-grade financial management to individuals. By combining strict double-entry bookkeeping principles with a modern, intuitive interface and cutting-edge AI integration, it offers the best of both worlds: correctness and convenience.

## Why LedgerOne?

- **True Double-Entry Bookkeeping**: Unlike simple expense trackers, LedgerOne ensures every transaction is balanced. Assets, Liabilities, Equity, Income, and Expenses are tracked with accounting precision.
- **Your Data, Your Control**: Built as a local-first application, your financial data remains on your machine. No cloud subscriptions, no data selling, complete privacy.
- **AI-Native Experience**: Integrated with the Model Context Protocol (MCP), allowing you to use AI assistants like Claude to query your finances, analyze spending, and even record transactions using natural language.
