# 基礎設定功能

## 1.1 註冊與啟動

### 功能描述

軟體啟用與註冊碼輸入，用於解鎖完整功能。

### 操作步驟

1. 下載並安裝我的記帳簿軟體
2. 執行軟體進入主畫面
3. 在註冊畫面輸入註冊通知的 Email 及註冊碼
4. 按下確定按鈕，待系統顯示註冊成功訊息

### 資料欄位

| 欄位名稱 | 類型 | 必填 | 說明                           |
| -------- | ---- | ---- | ------------------------------ |
| Email    | 文字 | 是   | 輸入註冊通知所用的電子郵件地址 |
| 註冊碼   | 文字 | 是   | 輸入對應的產品註冊碼           |

### 業務規則

- 註冊碼需與 Email 配對才能成功註冊
- 註冊成功後才能使用完整功能

## 1.2 建立帳號

### 功能描述

創建使用者帳號以區分及保護資料。多個帳號可用於區隔不同用途的記帳資料。

### 操作步驟

1. 定位至建立新帳號按鈕並點擊
2. 輸入帳號名稱與密碼
3. 點擊確定按鈕完成帳號建立

### 資料欄位

| 欄位名稱 | 類型 | 必填 | 格式限制     | 說明         |
| -------- | ---- | ---- | ------------ | ------------ |
| 帳號名稱 | 文字 | 是   | 中文或英文   | 建議使用英文 |
| 密碼     | 文字 | 是   | 僅限英數字元 | 用於帳號保護 |

### 業務規則

- 帳號名稱可以輸入中文，但建議使用英文
- 密碼則只能輸入英數字元組合
- 用戶可依需求建立不同的帳號來區隔資料
- 每個帳號的資料獨立儲存和管理

### 使用場景

- 家庭成員各自建立獨立帳號
- 區分個人帳戶與家庭帳戶
- 區分不同事業體的記帳

## 1.3 建立帳本

### 功能描述

建立記帳帳本並設定初始資金。一個帳號可以擁有多個帳本，用於區分不同時期或不同用途的記帳資料。

### 操作步驟

1. 按下建立按鈕
2. 輸入帳本名稱（可使用中文）
3. 建立成功後點擊開啟鍵
4. 輸入初始現金（目前身上的現金）
5. 確認後即完成帳本初始化

### 資料欄位

| 欄位名稱 | 類型 | 必填 | 格式限制 | 說明               |
| -------- | ---- | ---- | -------- | ------------------ |
| 帳本名稱 | 文字 | 是   | 支援中文 | 識別帳本用途       |
| 初始現金 | 數字 | 是   | 金額格式 | 目前身上的現金總額 |

### 業務規則

- 帳本名稱支援中文輸入
- 初始現金設定為必要步驟
- 初始現金代表帳本開始時的現金資產
- 每個帳本獨立管理科目和記錄

### 使用場景

- 年度帳本（如「2024年帳本」）
- 專案帳本（如「裝潢專案」）
- 短期帳本（如「日本旅遊」）

## 1.4 自訂科目

### 功能描述

自訂收支科目分類，以符合個人記帳需求。

### 操作步驟

1. 點擊指定按鈕開啟科目管理界面
2. 系統預設包含「現金」和「Equity」兩個科目（無法刪除）
3. 在紅色標示區域新增科目類別
4. 新科目自動加入上方科目列表

### 科目分類說明

#### 1. 資產 (Asset)

**定義**：可用金錢

**常用科目範例**：

- A-現金：手頭現金
- A-銀行存款：銀行帳戶餘額
- A-投資：股票、基金等投資
- A-應收帳款：借出的款項
- A-儲值卡：預付卡餘額
- A-XX股票：特定股票投資
- A-XX保險：儲蓄型保險

**特性**：收入會使得資產增加，支出會使得資產減少

#### 2. 負債 (Liability)

**定義**：需償還金錢

**常用科目範例**：

- L-信用卡：信用卡債務
- L-房貸：房屋貸款
- L-應付帳款：應償還的借款
- L-應付會錢：互助會待繳款項

**特性**：借入會使負債增加，還款會使負債減少

#### 3. 收入 (Income)

**定義**：收入會使得資產增加或負債減少

**常用科目範例**：

- I-薪資：工作收入
- I-投資收益：投資獲利
- I-利息收入：存款或投資利息
- I-其他收入：其他各類收入
- I-買賣外幣：外幣兌換收益

**特性**：增加淨資產的項目

#### 4. 支出 (Expense)

**定義**：支出會使得資產減少或負債增加

**常用科目範例**：

- E-餐飲費：飲食支出
- E-交通費：交通相關支出
- E-娛樂費：娛樂活動
- E-購物：一般購物
- E-網路購物：線上購物
- E-利息：貸款利息
- E-保險：保險費用
- E-其他支出：其他各類支出
- E-買賣外幣：外幣兌換損失
- E-投資損失：投資虧損

**特性**：減少淨資產的項目

### 科目資料欄位

| 欄位名稱 | 類型 | 必填 | 說明                              |
| -------- | ---- | ---- | --------------------------------- |
| 科目名稱 | 文字 | 是   | 科目的顯示名稱                    |
| 科目類型 | 選項 | 是   | Asset/Liability/Income/Expense    |
| 初始值   | 數字 | 否   | 科目的初始餘額（適用於資產/負債） |

### 業務規則

- 無法刪除兩個預設科目（現金、Equity）
- 新建科目必須選擇適當分類類別
- 科目名稱建議使用類型前綴（如 A-、L-、I-、E-）以便識別
- 每個帳本可以有不同的科目設定

### Equity 科目說明

- **用途**：權益調整科目
- **使用時機**：
  - 撿到錢等無來源收入
  - 記帳初期調整期初餘額
  - 特殊會計調整
- **特性**：系統預設科目，無法刪除

## 1.5 帳本管理功能

### 複製帳本

**用途**：備份或建立相似帳本
**操作**：選擇帳本後使用複製功能

### 更名帳本

**用途**：修改帳本名稱
**操作**：直接編輯帳本名稱
**注意**：若更名失敗，可能是權限問題，需參考故障排除章節

### 刪除帳本

**用途**：移除不需要的帳本
**操作**：選擇帳本後使用刪除功能
**警告**：刪除後資料無法復原，建議先備份
