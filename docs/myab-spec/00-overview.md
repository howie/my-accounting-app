# 系統概述

## 軟體簡介

**MyAB 我的記帳簿** 是一款個人記帳管理軟體，提供桌面版應用及行動裝置支援。

### 主要平台

- **Windows 版本**：MyAB 我的記帳簿
- **macOS 版本**：MacMoney 麥克錢包
- **網路版本**：支援 iPhone/Android 瀏覽器存取

- 軟體註冊與啟動
- 使用者帳號管理
- 帳本建立與管理

## 核心會計概念

### 科目分類系統

MyAB 採用四大科目分類：

1. **資產 (Asset)**
   - 定義：可用的金錢
   - 範例：現金、銀行存款、投資、應收帳款
   - 特性：資產增加代表財富增加

2. **負債 (Liability)**
   - 定義：需償還的金錢
   - 範例：借款、信用卡債務、應付帳款
   - 特性：負債減少代表財務狀況改善

3. **收入 (Income)**
   - 定義：使資產增加或負債減少的項目
   - 範例：薪資、投資收益、利息收入
   - 特性：收入會改善財務狀況

4. **支出 (Expense)**
   - 定義：使資產減少或負債增加的項目
   - 範例：餐飲費、交通費、購物、利息支出
   - 特性：支出會減少淨資產

### 預設科目

系統預設包含兩個無法刪除的科目：

- **現金**：基本資產科目
- **Equity**：權益科目，用於特殊調整

## 記帳教學涵蓋範圍

軟體支援以下 15 種記帳場景：

4. 借錢及還錢
5. 存錢及領錢
6. 遺失錢及撿到錢
7. 互助會（標會）
8. 儲值卡
9. 股票與基金
10. 商品退貨
11. 房貸
12. 保險費
13. 外幣記帳
14. 代墊款項
15. 定期付款

## 技術規格

### 資料庫限制

- 單一資料庫上限：30,000 筆記錄
- 備份檔案格式：.mbu
- 匯出格式：CSV、HTML

### 系統要求

- Windows 作業系統支援
- macOS 支援（MacMoney 版本）
- 網頁瀏覽器（網路記帳）

### 自動備份機制

- 軟體關閉時自動備份
- 每天保留一份最新備份
- 系統保留最近 10 個備份檔案
- 預設路徑：C:\Program files\MyAB\Backup\
