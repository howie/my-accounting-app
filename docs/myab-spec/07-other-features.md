# 其他功能

## 7.1 統一發票對獎

### 功能描述
記錄發票號碼並自動對獎，方便管理統一發票。

### 一、輸入發票號碼

#### 日常記帳時輸入

**操作方式**：
1. 在新增記帳記錄時
2. 在發票欄位輸入發票號碼
3. **只需輸入數字部分**，英文字母不必輸入
4. 系統自動儲存發票號碼

**範例**：
```
發票號碼：AB-12345678
輸入：12345678（僅數字部分）
```

#### 單獨輸入發票號碼

**使用時機**：
- 發票非自行購買獲得（如他人贈送）
- 發票日期與消費日不同
- 擁有未來日期的發票

**操作步驟**：
1. 進入發票管理功能
2. 透過清單另行輸入發票號碼
3. 日期以發票上的月份為主
4. 儲存發票資訊

**資料欄位**：
- 發票號碼（僅數字）
- 發票日期
- 金額
- 備註

### 二、統一發票對獎

#### 操作步驟

**步驟一：等待中獎號碼公布**
- 等財政部公布發票中獎號碼
- 通常為單數月份的25日

**步驟二：取得中獎號碼**
1. 進入發票對獎功能
2. 點選「取得號碼」按鈕
3. 系統自動取得本期的中獎號碼
4. 確認號碼已更新

**步驟三：開始對獎**
1. 按下「開始對獎」按鈕
2. 系統自動比對您的發票
3. 顯示對獎結果
4. 列出中獎發票（如有）

### 對獎結果顯示

#### 中獎資訊
- 發票號碼
- 中獎金額
- 獎項等級（特獎、頭獎、二獎等）
- 發票日期

#### 未中獎
- 顯示總共對獎筆數
- 確認對獎完成

### 業務規則
- 僅輸入發票號碼的數字部分
- 英文字母不需輸入
- 系統自動從網路取得中獎號碼
- 需要網路連線才能取得號碼

### 使用場景
- 每兩個月對一次獎
- 管理大量發票
- 避免漏掉中獎發票

### 注意事項
- 確保發票號碼輸入正確
- 定期執行對獎功能
- 中獎後記得兌獎

## 7.2 手機支援說明

### 功能概述
- 目前在 iPhone 與 Android 手機的支援以**網路版**為主
- 用戶連線到網站進行記錄
- 之後可以在 MyAB 軟體內下載網站上的資料

### 網路記帳網址
```
https://www.devon.riceball.net/mos/
```

### 使用前準備

#### 建立網路記帳帳號
1. 開啟網路記帳網址
2. 註冊新帳號
3. 設定使用者名稱和密碼

#### 設定科目資料
**建議在電腦上設定**：
1. 開啟 MyAB 桌面版軟體
2. 設定好所有科目
3. 將科目資料同步到網路記帳

### 手機使用步驟

#### 在 iPhone 或 Android 瀏覽器

**步驟一：開啟網路記帳**
1. 在手機瀏覽器中輸入記帳網址
2. 或使用書籤功能儲存網址

**步驟二：登入帳號**
1. 輸入網路記帳帳號
2. 輸入密碼
3. 登入系統

**步驟三：開始記帳**
1. 使用方法與桌面版網路記帳相同
2. 選擇科目
3. 輸入金額和明細
4. 儲存記錄

### iPhone 額外便利功能

#### 加入主畫面快捷

**操作步驟**：
1. 在 Safari 瀏覽器開啟網路記帳網站
2. 點擊分享按鈕
3. 選擇「加入主畫面」
4. 設定圖示名稱
5. 確認加入

**優點**：
- 像 App 一樣快速啟動
- 不需要每次輸入網址
- 方便日後快速存取

### 下載資料到桌面版

#### 操作步驟

**在 MyAB 桌面版軟體**：
1. 開啟要匯入資料的帳本
2. 點擊「下載記帳資料」功能
   - 注意：與匯入CSV在同一位置
3. 輸入網路記帳帳號和密碼
4. 選擇要下載的日期範圍
5. 系統顯示網路資料供預覽
6. 點擊「加入帳本」完成匯入

### 使用場景

#### 場景一：外出臨時記帳
- 外出購物時用手機立即記錄
- 回家後同步到桌面版
- 保持資料完整性

#### 場景二：辦公室記帳
- 在辦公室用網路版記帳
- 避免安裝桌面軟體
- 回家後下載到桌面版整合

#### 場景三：多人共用
- 家人各自用手機記帳
- 定期下載到家用電腦整合
- 統一管理家庭財務

### 網路記帳與桌面版的關係

#### 資料流向
```
手機/網路記帳 → 記錄暫存於網路
                ↓
         定期下載到桌面版
                ↓
           整合所有資料
```

#### 不是即時同步
- 網路記帳的資料暫存在伺服器
- 需要手動下載到桌面版
- 下載後成為桌面版的一般記錄

### 注意事項

#### 科目一致性
- 確保網路版和桌面版科目設定一致
- 避免下載時找不到對應科目

#### 定期下載
- 建議定期下載資料
- 避免累積過多未下載記錄
- 防止資料遺失

#### 網路連線
- 手機記帳需要網路連線
- 下載資料到桌面版也需要網路

## 7.3 網路記帳功能

### 功能描述
透過瀏覽器進行線上記帳，無需安裝軟體。

### 網路記帳網址
```
https://www.devon.riceball.net/mos/
```

### 主要功能

#### 基本記帳
- 新增收入/支出/轉帳記錄
- 編輯記帳資料
- 刪除記帳資料
- 查詢記帳記錄

#### 科目管理
- 新增自訂科目
- 編輯科目名稱
- 刪除科目

#### 報表檢視
- 簡易收支統計
- 科目餘額查詢

### 使用方式

#### 註冊與登入
1. 開啟網路記帳網址
2. 註冊新帳號（如未註冊）
3. 使用帳號密碼登入

#### 記帳操作
1. 選擇交易類型
2. 選擇科目
3. 輸入金額和明細
4. 儲存記錄

#### 資料下載
1. 在 MyAB 桌面版軟體
2. 使用「下載記帳資料」功能
3. 將網路資料匯入帳本

### 優點
- 跨平台使用（Windows、Mac、Linux）
- 不需安裝軟體
- 支援手機瀏覽器
- 資料儲存在雲端

### 限制
- 功能較桌面版簡化
- 需要網路連線
- 不支援自動備份
- 需手動下載到桌面版整合
