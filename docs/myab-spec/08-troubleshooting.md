## 8.1 資料出問題無法登入

### 問題描述

當記帳軟體資料損毀導致無法登入時的恢復方案。

### 自動備份機制

- 系統會自動保留最近 **10 個備份檔案**
- 備份在每次關閉軟體時自動執行
- 每天僅保留一份最新備份

### 情況一：能進入帳本選單

#### 操作步驟

1. 解除自動登入（如需要）
2. 使用帳號登入進入帳本選單
3. 點選「還原系統備份」按鈕
4. 選擇最近日期的備份還原
5. 若仍有問題，依序還原前一天的備份

### 情況二：無法進入帳本選單

#### 備份檔案路徑

```
C:\使用者\(Windows帳號)\AppData\Local\VirtualStore\Program Files (x86)\MyAB\Backup\(帳號)\
```

#### 操作步驟

1. 在登入介面點選「還原備份」按鈕
2. 前往上述備份資料夾
3. 選擇最近日期的備份還原
4. 如有錯誤，重複步驟選擇前一天的備份

## 8.2 高解析度螢幕文字模糊

### Windows 10 以前版本

1. 在 MyAB 執行檔上按右鍵選擇「內容」
2. 進入「相容性」標籤
3. 勾選「在高 DPI 設定時，停用顯示調整值」
4. 點擊確認

### Windows 10 及更新版本

1. 在執行檔上按右鍵選擇「內容」
2. 進入「相容性」標籤
3. 勾選「覆寫高 DPI 縮放行為」
4. 從下拉選單選擇「應用程式」
5. 儲存設定

## 8.3 資料無法儲存

### 非 Win10 系統

- 建議重新開機

### Win10 系統

- 完成更新後重新開機可解決問題

### 持續無法儲存

1. 備份資料
2. 移除軟體
3. 重新安裝至 D 槽等其他磁碟位置
4. 還原資料

## 8.4 軟體介面亂碼

### 操作步驟

1. 開始 → 設定 → 時間與語言
2. 選擇「其他日期、時間及區域設定」
3. 選擇「變更日期、時間或數字格式」
4. 進入系統管理分頁
5. 點選「變更系統地區設定」
6. 選擇「中文(繁體，台灣)」

### Windows 11 重要提醒

- 會出現「Beta 功能使用 Unicode UTF-8 提供全球語言支援」選項
- **務必不要勾選此項**
- 勾選會導致舊版軟體文字顯示異常

## 8.5 Windows 保護警告訊息

### 說明

- 這是微軟針對沒有發行者簽名的軟體的警示
- 並不代表軟體不安全

### 解決步驟

1. 點擊「其他資訊」
2. 在展開選項中選擇「仍要執行」按鈕

---

## 8.6 帳本名稱更名失敗

### 問題原因

預設路徑有權限問題造成更名失敗

### 解決步驟

1. 備份資料
2. 在控制台移除 MyAB
3. 重新安裝到其他位置（建議 D:\MyAB）
4. 在登入畫面還原備份

---

## 8.7 使用 Edge 瀏覽器無法下載軟體

### 下載階段

當 Edge 顯示「myab.exe 不常下載」警告時：

1. 點選右方的「...」選項
2. 選擇「保留」
3. 點擊「仍要保留」

### 執行階段

下載完成後出現 Windows 安全警告：

1. 點擊「其他資訊」
2. 選擇「仍要執行」以啟動程式

### 說明

這類警告並不代表軟體不安全，是微軟對未簽署軟體的標準安全機制。
